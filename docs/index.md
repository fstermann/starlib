# Starlib

Starlib is a DJ library management app. Organise your tracks, edit metadata, and manage artwork, all from a single interface.

<figure markdown="span" style="text-align: center;">
  ![Starlib](assets/images/screenshots/home.png){ width="90%" }
</figure>

## What can it do?

- **Library:** Two sources in one view — edit metadata on local files, or browse your SoundCloud likes and playlists (and other users' libraries under Discover)
- **Weekly Favorites:** Discover new tracks from followed artists, grouped by calendar week
